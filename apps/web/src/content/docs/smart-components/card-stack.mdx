## Props

| Prop        | Type   | Description                             | Default |
| ---------   | ------ | ---------------------------------       | ------- |
| className   | string | The class name for the component        | ""      |
| items       | `Array<{imageUrl: string; title: string; content: ReactNode}>` | Cards to render in the stack | ""      |
| offset       | number | Offset of the cards in the stack                           | ""      |
| scaleFactor | string | Scale of the cards in the stack                       | ""      |
